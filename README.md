# 🎲 Catan Web Implementation

An interactive, modern browser-based implementation of the popular board game **Catan**. This project is built entirely using clean client-side (Frontend) technologies without requiring a complex backend server. It features a responsive SVG hex board, dynamic resource management, an event log system, and goal-oriented computer AI opponents.

---

## 🚀 How to Run the Project Locally

Since this is a static website based on HTML, CSS, and JavaScript, **you do not need a specialized backend server (like Node.js or Python) to simply display the game**, but using a local server environment is highly recommended to avoid browser security restrictions (CORS) when loading dynamic files or modules.

Choose one of the following methods to run the project:

### 🛠️ Option 1: Using Visual Studio Code (Highly Recommended)
1. Open the project folder in **VS Code**.
2. Install the extension called **Live Server** (by *Ritwick Dey*).
3. Open the `index.html` file.
4. Right-click anywhere in the code editor and select **Open with Live Server** (or click the **Go Live** button in the bottom right status bar).
